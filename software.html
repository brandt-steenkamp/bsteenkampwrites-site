Software

Alongside writing, I build and contribute to software projects, primarily focused on accessibility and tools for blind or low-vision users.

Most of the work is open source and lives on GitHub.

Projects

anvill

anvill is an accessibility-focused toolchain for braille translation and publishing workflows, built using Swift, SwiftUI, and imported C libraries through a bridging header.

It is designed to integrate braille translation tools such as Liblouis into a streamlined workflow for producing accessible documents.

For now, it is macOS-only.

View the anvill repository on GitHub

Ice Cubes Contributions

Accessibility contributions to the Ice Cubes Mastodon client, including improvements to keyboard navigation and timeline interaction.

The work focuses on making timelines more usable for keyboard and screen reader users.
